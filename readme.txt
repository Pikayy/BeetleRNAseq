This folder contains all the files I used for the my first project during my phd. The project consists of De novo assembly results and DE analysis and each part is divided into smaller steps. 
allshfile: all the bash commands that I used, very important!!!
Multiqc: multiqc results of both calm and pus species
PCA: pca results generated by tmm normalized counts matrix
rsemmatrix: all the rsem raw counts (expected counts) matrix that will be used to feed to deseq2 or edgeR
trinotate: annotation results from trinotate
data:raw data
